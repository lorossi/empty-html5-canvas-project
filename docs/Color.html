Create a color by setting the value of its channels. Can be either RGB or HSL
Parameters:
Name Type Attributes Default Description
a number <optional>
 0 The value of the first channel (red for RGB, hue for HSL)
b number <optional>
 0 The value of the second channel (green for RGB, saturation for HSL)
c number <optional>
 0 The value of the third channel (blue for RGB, lighting for HSL)
d number <optional>
 1 The value of the alpha channel
rgb Boolean <optional>
 true If true, color will be interpreted as RGB. Otherwise, it will be interpreted as HSL
